# Hnefatafl AI Game

Artificial Intelligence course project implementing the Viking chess game Hnefatafl.

## Features
- Python GUI
- Prolog game logic
- Alpha-Beta pruning
- Human vs Computer mode
- Multiple difficulty levels

## Files
- `gui.py` → graphical interface
- `main.py` → game launcher
- `bridge.py` → Python-Prolog communication
- `game.pl` → game rules
- `controller.pl` → AI/game controller

## Run

```bash
python main.py
```
